# SoutShopping_eCommerce_FullStackProject
SoutShopping is an eCommerce Java full stack online shopping project developed using Spring boot with MySQL and core front-end technologies HTML5, CSS3 and JavaScript.<br/>
Project URL: https://soutshopping.herokuapp.com/

## Features
1. Separate login mechanism for admin and user
2. Admin Features:
   - Manage all products and categories
   - Cancel an order
   - Approve an order
   - Restrict user account
   - Manage pending orders
   - Manage successfully delivered products
   - Profile edit and update
3. User Features:
      - Manage orders history
      - Manage cart products
      - Review (rating) a product
      - Review ongoing orders
      - Can place new order
      - Profile edit and update
4. Other features:
   - Password stored in encrypted format in Database
   - User can't access pages secured for admin and admin can't access pages secured for users.
   - Role based login
   - OTP authentication for new user registration and forgot password
   - Secure payments using Razorpay Payment Gateway with almost all types of payment modes

## Technologies used
1. Java backend (Spring Boot, Spring Security, Spring Data JPA with Hibernate)
2. Thymeleaf Template Enginefor dynamic webpages
3. MySQL Database
4. Google Firebase
5. Apache Tomcat Server
6. Razorpay Payment Gateway
7. Java Mail API
8. HTML, CSS, JavaScript
